Для того, чтобы сделать возврат через личный кабинет необходимо:

  • Скачать заявление на возврат и заполнить его;
  • Сделать скан или фотографию заявления;
  • Выбрать или воспользоваться поиском заказа по номеру, прикрепить сканили фото заявления;
  • Заполнить обязательные поля;
  • Прикрепить дополнительные фотографии по товару при необходимости;
  • Оставить комментарий;
